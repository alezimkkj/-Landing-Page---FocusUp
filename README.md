# 🚀 FocusUp - Landing Page de Produtividade

Uma landing page moderna e responsiva para o **FocusUp**, um aplicativo focado em aumentar produtividade, organização e controle do tempo através da técnica Pomodoro.

Desenvolvido utilizando **HTML, CSS e JavaScript puro**, com abordagem **Mobile First** e foco em UI moderna.

---

## 📱 Preview

Landing page completa contendo:

- Hero Section
- Seção de Benefícios
- Timer Pomodoro funcional
- Depoimentos de usuários
- Call To Action
- Footer com contatos

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
- Mobile First Design
- Flexbox & Grid
- Media Queries
- Glassmorphism & Gradients UI

---

## 🎯 Funcionalidades

✅ Layout totalmente responsivo  
✅ Design Mobile First  
✅ Timer Pomodoro (25 minutos)  
✅ Botões de iniciar, pausar e reiniciar  
✅ Interface moderna com efeitos visuais  
✅ Animações e hover effects  
✅ Estrutura organizada por pastas
